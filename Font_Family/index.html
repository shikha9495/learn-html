Font

In metal typesetting, a font was a particular size, weight and style of a typeface. Each font was a matched set of type, one piece (called a "sort") for each glyph, and a typeface consisting of a range of fonts that shared an overall design

In typography, a typeface is a set of one or more fonts each composed of glyphs that share common design features. Each font of a typeface has a specific weight,style, condensation, width, slant, italicization, ornamentation, and designer or foundry.

A font is the combination of typeface and other qualities, such as size, pitch, and spacing. For example, Times Roman is a typeface that defines the shape of each character. Within Times Roman, however, there are many fonts to choose from different sizes, italic, bold, and so on.
